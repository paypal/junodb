# Gem Tasks
### Build gem
```
cd juno
rake build
```

### Install gem
```
cd juno
rake install
```

### Execute tests
```
cd juno
rake spec
```

### Execute rubocop linitng
```
cd juno
rake lint
```

### Update documentation
```
cd juno
rake yard:document
```
